## Installation

### 1. Clone the Repo

git clone https://github.com/SheftaAnis/taskManager.git
cd taskManager


### 2. Setup Backend
cd backend
npm install

create a .env file
PORT=5000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=yourpassword
DB_NAME=task_manager
JWT_SECRET=yourJWTSecret


### 3. Setup frontend
cd ../frontend
npm install
npm run dev
